# Tip Calculator

A simple bill splitter and tip calculator built with HTML, CSS, JavaScript, and Bootstrap.

## Run locally

Open `index.html` directly in your browser.

If you prefer a local server, use any static file server or IDE live preview tool from the project folder.

Then open the URL your tool gives, or simply open `index.html` in the browser.

## Notes

- No deployed URL yet.
- The app calculates total tip, total bill, and share per person.
- A light/dark theme toggle is available in the header.
